Step-by-Step Comparison

The same user action — clicking a link — triggers a very different sequence of events depending on whether speculation rules were active for that destination.

Standard Navigation (this page)
Click User clicks the link
DNS Browser resolves the hostname
TCP Open a new TCP / TLS connection
Req Send HTTP GET request
TTFB Wait for first byte from server
Parse Browser parses HTML + CSS + JS
Paint First paint — page becomes visible

All steps happen after the click. User waits through every one.

Prerender (Products / Technology)
Before Browser fetches HTML + parses + renders silently
Before Full composited frame held in memory
Click User clicks the link
Swap Browser activates the pre-rendered frame
Paint Page is already visible — no wait

All heavy work happened before the click. The user sees the page instantly.
